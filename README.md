# WordWild 📚 — The Ultimate Kids Reading Adventure

**WordWild** is a high-quality, gamified reading education app for children aged 3–8. Built with **Jetpack Compose** and modern Android best practices, it transforms learning into an immersive journey through phonics, vocabulary, and storytelling.

---

## 🚀 Key Features

*   **🔤 ABC Adventure:** Interactive alphabet exploration with high-quality audio.
*   **🃏 Word Match:** Build vocabulary by matching words to vibrant imagery.
*   **🎵 Rhyme Time:** Master phonetics by finding rhyming word pairs.
*   **📖 Story Builder:** Foster creativity by completing unique, engaging stories.
*   **✏️ Spell Blast:** Fast-paced unscrambling to master spelling.
*   **👁️ Sight Word Ninja:** Build reading speed with high-frequency word recognition.

---

## 🛠️ Tech Stack & Architecture

This project follows the **Modern Android Development (MAD)** guidelines:

| Technology | Purpose |
|---|---|
| **Kotlin** | Primary programming language. |
| **Jetpack Compose** | Fully declarative UI framework. |
| **Hilt** | Dependency injection for clean, testable code. |
| **KSP** | High-performance Kotlin Symbol Processing. |
| **Room Database** | Local data persistence for progress and stats. |
| **DataStore** | Modern preference storage for user settings. |
| **Google Fonts API** | Optimized, downloadable typography (Nunito). |
| **Material 3** | Latest Android design system. |
| **Coroutines & Flow** | Reactive, asynchronous programming. |

## 📦 Getting Started

### Prerequisites
*   **Android Studio Ladybug** (or newer)
*   **JDK 17**
*   **Target SDK 35** (Android 15)

### Setup
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/prabhavithreddy/WordWild.git
    ```
2.  **Open in Android Studio:** Select the `WordWild/` folder.
3.  **Build & Run:** Click the "Run" button to deploy to an emulator or physical device.

---

## 🎯 Publishing Details

*   **App Name:** WordWild
*   **Package Name:** `com.readikids.app`
*   **Version:** 1.0.1 (Code 2)
*   **Minimum SDK:** 26 (Android 8.0)
*   **Target SDK:** 35 (Android 15)

### Generating a Release Bundle
1.  Go to `Build > Generate Signed Bundle / APK...`
2.  Select **Android App Bundle (.aab)**.
3.  Follow the wizard to sign the bundle with your release keystore.
4.  Upload the generated `.aab` file to the [Google Play Console](https://play.google.com/console/).
